Cosm API Documentation

  • Index of Functions

cosm.h

  • cosm.h - Main header which includes all of the CPU/OS and utility layer headers, defines, build settings, as well as the main self-test function. Users of the Cosm libraries simply include this file.

CPU/OS Layer

  • os_file.h - File and directory functions.
  • os_io.h - Input/output and string functions.
  • os_math.h - Math functions.
  • os_mem.h - Memory functions.
  • os_net.h - Network functions.
  • os_task.h - Process, thread, CPU, mutex, and clock functions.

Utility Layer

  • bignum.h - Big number functions.
  • buffer.h - Memory buffer functions.
  • config.h - Program config file functions.
  • email.h - Email functions.
  • http.h - HTTP protocol functions.
  • language.h - Human language functions.
  • log.h - Log file functions.
  • security.h - Data signing, encryption, hash, and random functions.
  • time.h - Time, calendar, and timing functions.
  • transform.h - Transform functions.

2nd Party Libraries

  • bzip2/bzip2.h - bzip2 compression functions for use as transforms.

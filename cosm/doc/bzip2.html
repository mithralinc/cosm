bzip2 Library Wrappers

  • COSM_BZIP2_COMPRESS
  • COSM_BZIP2_DECOMPRESS

COSM_BZIP2_COMPRESS

Syntax

 
#include "cosm/transform.h"
#include "cosm/bzip2/bzip2.h"

Description

Define for use in Transform functions. params = u32 to compression level. Output to required next_encoder is the data compressed with bzip2.

COSM_BZIP2_DECOMPRESS

Syntax

 
#include "cosm/transform.h"
#include "cosm/bzip2/bzip2.h"

Description

Define for use in Transform functions. params = none. Output to required next_encoder is the bzip2 data decompressed.
